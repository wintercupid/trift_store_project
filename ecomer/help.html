Navigation Features

Use the search icon in the top navigation to find specific products. You can search by:

  • Product name (e.g., "Summer Dress")
  • Category (e.g., "Men's T-shirts")
  • Color or style

Access your cart from any page using the cart icon in the top navigation.

From the cart you can:

  • Review selected items
  • Adjust quantities
  • Proceed to checkout
  • Save items for later
View Cart

The user icon provides access to:

  • Order history
  • Saved addresses
  • Wishlists
  • Account settings
You'll need to create an account or log in to access these features.
